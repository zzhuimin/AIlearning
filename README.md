# CSV 编码转换器

自动检测 CSV 文件编码并转换为 UTF-8，解决中文乱码问题。

## 安装

### 方式一：Kimi Claw 对话安装
1. 打开 Kimi Claw
2. 上传 `SKILL.md` 文件
3. 说："安装这个 Skill"

### 方式二：本地 OpenClaw
```bash
mkdir -p ~/.openclaw/skills/csv-encoding-converter
cp SKILL.md main.py requirements.txt ~/.openclaw/skills/csv-encoding-converter/
# 重启 OpenClaw 或刷新技能
